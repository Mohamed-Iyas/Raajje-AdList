# Raajje AdList

An ad-blocking filter list for Maldivian websites.

## What is this?

This repository contains a filter list (`filter.txt`) that can be used with ad blockers like Adblock Plus and uBlock Origin. The filter list contains rules to block advertisements on various Maldivian websites, providing a cleaner and faster browsing experience.

## How to use

To use this filter list, you need to add the following URL to your ad blocker's filter list subscriptions:

```
https://raw.githubusercontent.com/Mohamed-Iyas/Raajje-AdList/master/filter.txt
```

### Adblock Plus

1. Go to **Options > Advanced > Filter lists**.
2. Click **+ Add a new filter list**.
3. Select **Add a custom filter list**.
4. Enter a name for the filter list (e.g., "Raajje AdList").
5. Enter the URL: `https://raw.githubusercontent.com/Mohamed-Iyas/Raajje-AdList/master/filter.txt`
6. Click **Add filter list**.

### uBlock Origin

1. Go to **Options > Filter lists**.
2. Under **Custom**, click **Import...**.
3. Enter the URL: `https://raw.githubusercontent.com/Mohamed-Iyas/Raajje-AdList/master/filter.txt`
4. Click **Apply changes**.

## How to contribute

Contributions are welcome! If you find an ad on a Maldivian website that is not blocked by this filter list, please [create an issue](https://github.com/Mohamed-Iyas/Raajje-AdList/issues) or submit a pull request.

When submitting a pull request, please ensure that your changes are consistent with the existing filter list format.
